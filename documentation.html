ChocolateChip-UI together with ChocolateChip.js comprise a powerful framework for creating Web apps. ChocolateChip.js is the foundation upon which the other parts rests. ChocolateChip-UI consists of two files: ChUI.js and ChUI.css. It uses a special markup language called WAML. Read the documentation for all four of these by choosing from the menu on the left.
