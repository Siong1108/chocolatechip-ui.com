Data Binding

 

Knockout.js

Knockoutjs is a powerful, client-side data-binding engine. It allows you to create view models and then bind your markup to them using declarative expressions. When the view model changes, the markup updates automatically, no DOM manipulation required on your part. Knockoutjs can be used with static markup to create dynamic temlates, or it can be used with an actual template engine.

ChocolateChip-UI includes an example of using Knockoutjs in the 'data-binding' folder in the examples.

JSViews

If you're already using JSRender as your template needs, you can combine that with JSViews to achieve client-side data binding on your templates similar to what Knockoutjs does.
