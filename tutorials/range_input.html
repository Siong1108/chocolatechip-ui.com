Range Input

ChocolateChip-UI uses the HTML range input to provide a stylable range control. This is supported in Android 4.0 and iOS 5. Using CSS it's easy to style the range input to look like whatever you branding needs might be.

Example:

<input type="range" id="rangeControl1" value="0" min="0" max="100">

Setting a Default Value:

You can set a default value for when the range input is loaded by simply giving it a value:

<input type="range" id="rangeControl1" value="25" min="0" max="100">

You can also set a step value so that the range input increases and decrease at set values, such as by 10.

<input type="range" id="rangeControl1" value="25" min="0" max="100" step="10">

Tracking Changes:

To track changes in the value of the range input, just listen for a change event on the input. The following example uses jQuery/Zepto:

$('#rangeValue1').html($('#rangeControl1').val());
   $('#rangeControl1').on('change', function() {
      $('#rangeValue1').html($(this).val());
});

To modify the appearance of the range input is fairly easy. You just need to adjust dimensions, such as height and width, and change values for the gradient that colors the track and the thumb. Please refer to the documentation for ChUI.CSS for details on how to do this.
