Titled Lists

Titled lists are an additional type of table view. These are a type of navigation list. Selecting an item in the list will take the user to the next screen. Basically, a titled list is just a single tableview broken up into subsections by tableheaders as titles. The tableview tablecells are like a typical navigation list consisting of tablecells with href values for where the cells lead to. To turn such a navigation list into a titled list, just put ui-kind=”titled-list” on the tableview.

Example:

<tableview ui-kind="titled-list">
    <tableheader>Fruits</tableheader>
    <tablecell href="apple_detail">

    </tablecell>
