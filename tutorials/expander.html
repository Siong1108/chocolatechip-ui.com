Expander

ChocolateChip provides the expander as a way to allow collapsing and expanding a vertical section of an app. The look and behavior of the expander were inspired by the expander in WebOS. To implement an expander is really easy. Just put an expander tag in your app with a panel tag inside that. Your content goes inside the panel inside the expander. ChocolateChip will take care of styling the header, etc. The expander has a label that toggles between "Open" and "Close".

To change the defaults of the expander you can use the Element.UIExpander() function. This takes an object literal of various possible values:

  • status: "expanded" or "collapsed", the default is "expanded".
  • title: the label of the expander when it's collapsed, the default is "Open".
  • altTitle: the label of the expander when it's expanded, the default is "Close".

Example:

$(function() {
    var opts = {
        status: "collapsed",
        title: "Show",
        altTitle: "Hide"
    };  
    $("expander").UIExpander(opts);
});

<expander>
    <panel>
        <tableview ui-kind="grouped">
            <tablecell>
                <celltitle>Breakfast</celltitle>
            </tablecell>
            <tablecell>
                <celltitle>Lunch</celltitle>
            </tablecell>
            <tablecell>
                <celltitle>Dinner</celltitle>
            </tablecell>
        </tableview>
    </panel>
</expander>

See the documentation for ChUI.css to learn how to customize the look of the expander.
