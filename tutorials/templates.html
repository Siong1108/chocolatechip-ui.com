Templates

ChocolateChip-UI does not have a built-in templating solutions. However, there are a number of existing templating solutions that can be used with ChocolateChip-UI. Templates are used to output markup created based on JSON data retrieved by an AJAX request. This is especially useful when deal with large data sets being retrieved dynamically based on choices the user makes in an app.

There are so many templating solutions. As a matter of fact, you may already have a preferred templating solution. I've encluded some possible candidates here. One thing to bear in mind is speed. Not all template systems are equal. Some are faster and some are much slower. If you are going to be doing a lot of dynamic templating and using large amounts of data, you'll want to make sure you are using a fast template engine. You might want to examine the results of performance tests to get a clearer picture.

To help you get a better idea how to use templates with ChocolateChip-UI, you will find an example of Handlebars in a templates folder in the examples folder.

 

Dust

Dust is fast, efficient and flexible. There's a reason this is at the top of the list.

 

doT

This library was designed from the ground up to be fast, which it is.

 

Pure

Pure is a simle and faste templating solution to provide logicless templating. It uses an unobtrusive approach to templating.

 

JSRender

If you've used jQuery Templates in the past, this is their success. The author of jQuery Templates has created this as a more versatile and faster alternative to jQuery Templates. Using the 'no-conflict' flag, you can use it with ChocolateChip.js and Zepto.

 

Handlebars

Handlebars is a popular templating solution to provide logicless templating. It's based on the populare Mustache templating engine. In fact it is a superset of Mustache.

 

ICanHaz

Despite the quirky name, ICanHaz is a powerful and flexible templating solution.

 

Underscore

Underscore is the Swiss knife of utility functions and includes a template system which is the default for the popular Backbone.js framework.
