Dark Holo Theme

 

Reimaged for Android

The Holo theme for ChocolateChip-UI turns your Web app into a sleek-looking rendition similar to the dark Holo theme of Android 4.0 and later. Like Holo, it uses shades of slate for base elements with subtle blue tints to highlight hovers, focus, etc. By default this theme also uses Google's Roboto font. The images below show the Holo theme in action.

To implement the Holo them for Android, just make sure that the references to chui.css, chui.desktop.css and chui.js have their android postfixes (the desktop version gets loaded automatically when on a desktop operating system):

Example:

<script src="../../chui/chui.android.js"></script>
<link rel="stylesheet" href="../../chui/chui.android.css">
